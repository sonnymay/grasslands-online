# Grasslands Online

A browser-based idle/strategy game set in a vibrant grasslands world.

## Overview

Build and manage your grasslands settlement, level up your hero, and battle enemies from classic games like Digimon and Ragnarok Online.

## Features

- Idle progression with active build-mode
- Summon and awaken shadow soldiers using AI-generated art
- Explore grassland biomes with dynamic encounters
- HTML loading bar and background music system

## Tech Stack

- **Language:** JavaScript
- **Deployment:** Vercel
- **CI/CD:** GitHub Actions
